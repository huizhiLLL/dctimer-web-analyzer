# dctimer-web-analyzer

DCTimer 导出数据库（`.db` / SQLite）纯前端分析网站。

当前阶段：前期方案与数据结构梳理。

## 目录

- `docs/TYPE-MAPPING.md`：DCTimer `sessiontb.type` 编码规则与项目映射
- `docs/PARSER-DESIGN.md`：前端解析器设计草案
- `docs/DATA-FLOW.md`：统一业务模型到统计聚合模型的数据流设计
- `docs/UI-DESIGN-BASELINE.md`：移动端友好、简约高级感的前端设计基线

## 目标

- 纯前端、本地解析、不上传数据
- 先按分组筛选，再生成年度总结式汇总报告
- 偏作品型、游戏化、可分享
